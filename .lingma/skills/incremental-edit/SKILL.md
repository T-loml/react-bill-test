# Skill: Incremental Code Modifier

## 核心目标
确保 AI 修改代码时“基于现有代码增量修改”，禁止重写

---

## 修改原则（非常重要）

### ❌ 禁止行为
- 重写整个文件
- 删除未要求修改的逻辑
- 重构无关代码结构
- 覆盖用户已修改内容

---

### ✅ 必须遵守

- 只修改“用户明确要求的部分”
- 基于 diff / patch 思维
- 保留所有已有逻辑
- 不改变文件结构（除非明确要求）

---

## 修改策略

AI 必须遵循：

1. 先理解现有代码
2. 找出最小修改范围
3. 局部修改
4. 保持其余代码完全不变

---

## 输出要求

- 输出修改后的代码片段或完整文件（仅包含必要修改）
- 不解释
